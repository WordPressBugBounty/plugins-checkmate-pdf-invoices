Create custom PDF Invoices and Packing Slips for WooCommerce. Includes a Visual Template Editor, HPOS support, Bulk Actions, and Email Attachments.

== Description ==

**Checkmate PDF** is the modern solution for WooCommerce invoicing. Stop struggling with code snippets—customize your documents using our visual drag-and-drop editor.

Automatically generate and attach professional PDF Invoices to your WooCommerce emails. Fully compatible with High-Performance Order Storage (HPOS).

### 🚀 Key Features

*   **Visual Template Editor**: Customize your invoice layout, colors, and fonts without writing a single line of code.
*   **Automatic Email Attachments**: Automatically attach PDF invoices to "Order Completed" or "Processing" emails.
*   **Bulk Downloads**: Select multiple orders in the admin panel and download all invoices as a single ZIP file.
*   **HPOS Compatible**: Built for the future of WooCommerce with full High-Performance Order Storage support.
*   **Guest Access**: Allow customers to download invoices directly from the "Order Received" page without logging in (via secure unique keys).
*   **Packing Slips**: Generate packing slips for your fulfillment team alongside invoices.
*   **Numbering Systems**: flexible invoice numbering options.

### 🛡️ Developer Friendly

*   **Isolated Dependencies**: Bundles a prefixed version of `Dompdf` to ensure zero conflicts with other plugins using different versions of PDF libraries.
*   **Clean Code**: Built with modern PHP standards and strict typing.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the WordPress Plugins screen.
3. Go to WooCommerce > PDF Invoices (or the Checkmate menu) to configure your template.
